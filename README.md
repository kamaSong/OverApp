# OverApp — 单词学习APP

毕业设计项目 | Android + Java + SQLite


## 项目简介

独立开发的 Android 英语单词学习 App，涵盖词库管理、智能复习、拍照取词、学习追踪等功能。

**开发周期**：2023.11 - 2024.05
**参考书籍**：《第一行代码——Android》（郭霖 著）

## 技术栈

- Java
- Android
- SQLite（LitePal ORM）
- OkHttp
- Gson
- WorkManager

## 核心功能

- **智能复习**：自研四阶段艾宾浩斯遗忘曲线算法，根据掌握程度动态调整复习间隔
- **每日一图**：接入必应每日壁纸 API，启动页展示高清壁纸 + 每日一句
- **数据库设计**：11 张关联表，遵循第三范式，支持生词本多对多分类
- **词库管理**：解析有道词库嵌套 JSON（5000+ 单词），批量拆分写入 4 张表
- **学习追踪**：每日打卡、学习时长统计、日历可视化
- **拍照取词**：接入百度 OCR API，拍照识别单词并导入学习列表
- **游戏化学习**：单词速过（自动轮播）、单词连连看（配对消除）
- **定时提醒**：AlarmManager + WorkManager 双重提醒
- **深色模式**：适配日夜间主题切换

## 算法设计

四阶段艾宾浩斯复习算法：

| 阶段 | 触发条件 | 状态字段 |
|------|----------|----------|
| 新学 | 每日分配新词 | `isNeedLearned` |
| 及时复习 | 刚学完的单词 | `justLearned` |
| 浅度复习 | 掌握程度 < 10 | `masterDegree` (0-10) |
| 深度复习 | 掌握程度 = 10 | `deepMasterTimes` (0-3) |

- 答对：`masterDegree + 2`
- 超时未复习：`masterDegree - 2`（降级）
- 深度复习间隔：4天 → 3天 → 8天 → 完全掌握

## 数据库表结构

| 表名 | 作用 |
|------|------|
| Word | 单词本体（拼写、音标、掌握程度） |
| Interpretation | 一词多义（词性 + 释义） |
| Sentence | 例句 |
| Phrase | 短语 |
| User | 用户信息（词汇量、金币） |
| MyDate | 每日学习记录 |
| LearnTime | 学习时长 |
| WordFolder | 生词本 |
| FolderLinkWord | 生词本多对多关联 |
| UserConfig | 用户配置 |
| DailyData | 每日壁纸 |
